# v1.65 HUD 信息密度整理

## 版本号
v1.65

## 版本分配依据
v1.64 已统一反制回放/复核卡。本轮按文档建议整理顶栏状态条与地图 HUD 信息密度，提升战区指挥台可读性与窄屏适配。

## 目标
1. 压缩 `StatusChip` / `StatusStrip` 视觉噪音，保持全部关键战况字段。
2. 收紧 `MapCampaignHUD` / `MapActionHUD` / `MapHudMetric` / `MapQuickCommandButton` 间距与固定宽度，减少遮挡地图。
3. `MapToolbar` 副标题避免与顶栏/消息条完全重复长消息。
4. 快捷命令按钮最小点击高度提升到约 36–44pt。
5. 不改 `GameState` 数据与命令；仅 `ContentView`。
6. 文档同步 + 云端 Actions。

## 非目标
- 不重做侧栏、不改规则、不本地 Xcode。
- 不做完整触控手感大改（可在后续 v1.66）。

## 验收
- HUD 更紧凑且字段不丢。
- Actions success + Agent C artifact 核对。
