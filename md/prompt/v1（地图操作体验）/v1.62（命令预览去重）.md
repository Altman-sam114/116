## 背景

地图 HUD 内联命令预览与侧栏聚焦命令预览各自实现 icon/title/detail/route/fire/post-move/safe-engagement/accentColor，文案略有差异但逻辑重复，后续改规则文案易漏改一侧。

## 目标

1. 抽出共享只读 helper（不改 `GameState`/`GameModels`）。
2. 两面板共用 accent、icon、威胁名去重、路线摘要核心、火力暴露核心、移动后攻击核心、安全接敌建议核心。
3. 保留各自呈现密度：内联更短，侧栏更完整并含“右键”指引与攻击位补充。
4. 同步 README / flow / flowchart / test / prompt README / update_log。

## 非目标

- 不改命令执行、预览生成规则。
- 不合并两个 View 成单一布局。
- 不跑本地 Xcode；云端 Actions 验证。
